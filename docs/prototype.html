What this file is
A clickable prototype of a reusable subdivision feasibility + capital-strategy tool, built as a static web app. Pre-loaded with the real Clevedon 8-lot deal and current NZ financing settings. The Confirm chat both verifies the valuation and leads the owner to articulate the options — sell all, or keep some for long-term growth — then the engine models each on cash, serviceability and a 10-year horizon. The questions this answers
Cash-on-cash: freshCash = new money Ed injects after recycling section sales, drawing serviceable debt and repaying the mortgage; equityLocked = illiquid equity in the rentals. Fund-ability today: NOI − debt×actualRate; negative ⇒ annual top-up. Opportunity cost: 10-yr net-worth CAGR from equityToday = asIs − mortgage vs the hurdle.
